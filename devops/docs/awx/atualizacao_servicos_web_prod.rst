Atualização de serviços WEB Produção
====================================

Esta pagina aborda o passo a passo para a Atualização de serviços Web - nuvem

Ao rodar este template a ultima tag liberada do serviço é verificada e caso ela seja maior que a versão atual em produção, irá atualizar imediatamente o serviço na nuvem.

#. Acessar os templates do AWX

#. Selection o ``Workflow Job Template`` "`Atualização de serviços Web - nuvem <https://awx.korp.com.br/#/templates/workflow_job_template/357/details>`_"

#. Clicar em ``Launch``

#. Selecione o valor para o campo:

    - Selecione os serviços a serem atualizados

#. Execute a tarefa

.. note:: 

    Esse workflow precisa ser Aprovado, isso deve ser feito pela qualidade.
